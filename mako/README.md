# Mako Configuration

This is a minimal Mako config, a Wayland notification daemon.

## Installation

Place this file as `~/.config/mako/config`.  

## Dependencies

- `mako` – the notification daemon itself (Wayland only).
- `ttf-jetbrains-mono` – or any Nerd Font variant of JetBrains Mono.  
  If you use a different font, adjust the `font` line accordingly.

## Notes

- Mako does not work under X11; use it with Sway, dwl, River or another Wayland compositor.
